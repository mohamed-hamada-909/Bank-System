# 🏦 Bank Management System After Update (OOP)

A console-based **Bank Management System** built with **C++** using full **Object-Oriented Programming (OOP)** principles — including classes, inheritance, abstract classes/interfaces, and encapsulation.

The project simulates a real banking environment with client management, transactions, secure login, user permissions, and activity logging.

---

## ✨ Features

### 👤 Client Management
- Add, update, delete, find, and list clients
- View total balances across all clients

### 💰 Transactions
- Deposit and withdraw funds
- Transfer money between clients
- Transaction log kept for all transfers

### 🔐 Login & Security
- Secure login/logout system with permission levels
- Account lockout after 3 failed login attempts
- Passwords encrypted before being stored in file
- Login attempts recorded in a log file

### 🛠️ User Management (Admin)
- Add, update, delete, find, and list system users
- Assign specific permissions per user (role-based access)

### 🖥️ User Interface
- Full menu-driven console UI (Main Menu → Transactions, Manage Users, Login)
- Each screen displays current date and logged-in user

---

## 💱 Bonus Module: Currency Exchange

A separate OOP module simulating a currency exchange system:
- Add and manage currencies with exchange rates
- Search and update currency rates
- Currency calculator to convert between currencies

---

## 🧠 OOP Concepts Applied
- Classes & Objects
- Encapsulation (private data, public interfaces)
- Inheritance & Abstract Classes / Interfaces
- Static members & static binding
- File handling (storing clients, users, and logs)

---

## 🚀 How to Run
1. Clone the repository
2. Open the project in your C++ IDE (e.g., Code::Blocks, Visual Studio)
3. Build and run the executable
4. Login with a registered user to access the system
